PassGEN 🔐

Gerador de senhas aleatórias desenvolvido com HTML, CSS e JavaScript puro.<br>
O projeto permite criar senhas personalizadas com letras maiúsculas, minúsculas, números e símbolos, além de copiar a senha diretamente para a área de transferência.<br>
<br>
🚀 Funcionalidades <br>
    Gerar senhas aleatórias<br>
    Escolher o tamanho da senha<br>
    Incluir:<br>
    Letras maiúsculas<br>
    Letras minúsculas<br>
    Números<br>
    Símbolos<br>
    Copiar senha com 1 clique<br>
    Feedback visual ao copiar senha<br>
<br>
🛠 Tecnologias utilizadas<br>
    HTML5<br>
    CSS3<br>
    JavaScript (Vanilla JS)<br>
<br>
🧠 Lógica utilizada<br>
<br>
  querySelector para manipulação do DOM<br>
  Eventos com addEventListener<br>
  Arrays e flat()<br>
  Geração aleatória com:<br>
  Math.floor(Math.random() * chars.length)<br>
  Clipboard API:<br>
  navigator.clipboard.writeText()<br>
<br>
